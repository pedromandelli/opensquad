# Domain Framework - VascularCare Carousel System

## Framework de Producao (fim a fim)

1. **Tema e contexto clinico**
   - definir tema principal do carrossel
   - explicitar dor/duvida do paciente
   - mapear qual pilar D.N.A. sera enfatizado

2. **Escolha de angulo editorial**
   - selecionar formato narrativo (Editorial, Problema->Solucao, etc.)
   - definir promessa do cover
   - definir progressao logica dos slides

3. **Construcao textual**
   - criar headlines fortes e objetivas
   - criar supporting text com clareza e precisao
   - manter 40-80 palavras por slide
   - fechar com CTA especifico

4. **Orquestracao visual**
   - aplicar identidade premium (tipografia, paleta, espacos)
   - manter consistencia de header/footer
   - alternar ritmo visual de fundos
   - garantir legibilidade mobile

5. **Revisao e aprovacao**
   - validar ciencia, marca e performance Instagram
   - aplicar scorecard com criterios objetivos
   - aprovar ou iterar com ajustes acionaveis

## Template de Arquitetura de Slide

- **Header:** data + volume + assinatura medica
- **Corpo:** headline (claim) + supporting text (explicacao)
- **Elemento de destaque:** palavras-chave em cor accent
- **Footer:** selo/assinatura VascularCare

## Regras de Prioridade

1. Precisao clinica > estilo
2. Clareza > jargao
3. Valor pratico > discurso institucional
4. Swipe progression > slides isolados
5. CTA util > CTA generico
